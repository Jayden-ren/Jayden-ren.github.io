环境配置

  • 推荐使用anaconda进行环境之间的隔断. 这样可以创造一个个单独的环境

使用anaconda

  1. 下载anaconda,并进行安装.安装时选择自动配置环境即可
  2. 下载完成后开始菜单中会出现Anaconda Powershell Prompt (Anaconda)即anaconda的命令行
  3. 打开后与win10自带的cmd命令行如出一辙。 有一点不同是命令行行首会出现(base)字样。 意思是当前处于base环境下。
conda list    # 查看当前环境下安装了哪些环境
conda env list  # 查看当前电脑下有哪些环境，并列出
  1. 新建环境
conda create -n ProjectName    # 新建一个名为ProjectName的文件，加到conda环境中。默认路径是安装anaconda的安装目录中的envs文件夹内
conda install python=3.7.4
conda install opencv=3.4.2
conda install numpy     # 指定版本，如果不指定默认安装最新版
  1. 使用Clion或者Pycharm配置。两者操作一模一样， 界面上方有个小扳手符号，打开后add新环境变量。选择conda环境中的已存在环境。这里需要配置两个值上面的是python.exe文件。下面的是conda.exe 路径。

1. 测试版本

import cv2

2. 加载图片

import cv2 as cv
import numpy as np

src = cv.imread("img/lena.jpg")
cv.imshow("src", src)
cv.waitKey()
  • cv.imread() 双引号里的是读取文件的位置。上面填的是相对位置，相对py文件的位置。也可以填绝对位置，但建议填相对位置，这样在打包程序时不会找不到图片
  • cv.imshow() 展示src这个图片。创建一个窗口，窗口标题为src，可以取中文标题
  • cv.waitKey() 等待关闭，括号里的参数填数字（不等于0）则单位是毫秒。填单个字符则返回该字符对应的ascII码
